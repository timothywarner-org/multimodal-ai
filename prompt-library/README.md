# Prompt Library

Curated prompts for IT-focused roles. Each file is copy/paste-ready for fast experiments in chat
playgrounds, Copilot Studio, or Azure AI Foundry.

## How to use

* Start with the **System starter** to set tone and boundaries
* Grab 1-2 **Task prompts** and adjust nouns/URLs/IDs
* Use **Follow-ups** to tighten outputs (format, scope, safety)

## Roles covered

* Developers: `prompt-library/dev.md`
* DevOps/Platform: `prompt-library/devops.md`
* Solutions consultants: `prompt-library/solutions-consultant.md`
* Data science/analytics: `prompt-library/data-science.md`
* IT operations/SRE: `prompt-library/ops.md`
* Security: `prompt-library/security.md`
