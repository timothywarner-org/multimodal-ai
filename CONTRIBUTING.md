# Contributing

Thanks for helping improve this documentation repo.

## Scope

* Documentation only: Markdown tutorials, prompt libraries, and resources for multimodal AI
* No application code expected

## How to contribute

1. Fork/branch; keep changes focused and small.
1. Follow Markdown rules in `.markdownlint.json` (100-char lines, ATX headings, asterisk lists).
1. Run lint before raising PR: `npx markdownlint-cli2 "**/*.md" --config .markdownlint.json --fix`.
1. Check links if you add many URLs: `lychee --recursive --exclude-mail --timeout 10 *.md`.
1. Use conventional commits (e.g., `docs: add copilot tutorial`).
1. Open a PR with a brief summary and any testing notes.

## Style tips

* Write for mixed skill levels; prefer concise, stepwise guidance.
* Include copy/paste-ready prompts or commands.
* Cite official docs where possible.
